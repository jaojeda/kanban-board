# Task Board PRD

## Goal

A local-first Kanban board. No backend. Data persists in localStorage.

## Columns

Three fixed columns: To Do, In Progress, Done

## Cards

Each card has: title (required), description (optional), priority (low/medium/high)

## Interactions

- Add card to any column
- Move card between columns
- Delete card
- Edit card title/description

## Out of Scope (for now)

- User auth
- Multiple boards
- Due dates
